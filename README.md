# Organizer

## Description
Used to organize roms sets:
1. USA
2. Proto, Demo, Alpha, Beta and Samples
3. Other Regions
4. Keep just the last revision of a rom

## Usage
Just put the script inside rom folder and execute:
`python3 rom_organizer.py`
